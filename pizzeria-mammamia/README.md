# Pizzería Mamma Mía 🍕

Proyecto realizado en **React + Vite** como parte del Desafío 1.  
La aplicación simula la página web de una pizzería, con navegación entre secciones y componentes reutilizables.

## 🚀 Tecnologías utilizadas
- React 19
- Vite 8
- React Router DOM
- ESLint

## 📂 Estructura del proyecto
- `src/components` → Navbar, PizzaCard
- `src/pages` → Home, Menu
- `src/assets` → imágenes y estilos
- `App.jsx` → configuración de rutas principales
